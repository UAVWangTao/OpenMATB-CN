# 这是一个实验介绍语模板
# 实验介绍语必须通过简单的HTML标签语法进行格式化

# 以下元素完全支持：
#    B BLOCKQUOTE BR CENTER CODE DD DIR DL EM FONT H1 H2 H3 H4 H5 H6 I IMG KBD
#    LI MENU OL P PRE Q SAMP STRONG SUB SUP TT U UL VAR

# 您可以通过<newpage>标签将实验介绍语分为多个部分
# OpenMATB中有三个常用标签：<h1>、<h2>、<p>，使用这三个标签会自动将文本渲染为预定义的大小、粗细等等
